## Prerequisites

- Node.js 18+
- npm or pnpm

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment:
   Copy `.env.example` to `.env` (optional, defaults to localhost:8000).

## Development

Start the development server:
```bash
npm run dev
```
Open http://localhost:5173 (or port shown in terminal).

## Build

Build for production:
```bash
npm run build
```

## Testing

Run unit tests:
```bash
npm test
```

## WSL Tips

If running on WSL:
- Ensure you are running in the WSL terminal.
- If browser doesn't open automatically, click the link in the terminal.
